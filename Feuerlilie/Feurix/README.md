# ?? Feurix
> *Огненное ядро Rust-инструментов в структуре Wasserfall/Feuerlilie*

Feurix — это модуль, созданный для автоматизации, прозрачности и инженерной строгости.
Он легко интегрируется в рабочий процесс и служит фундаментом для будущих Rust-компонентов.

## ?? Цели
- Обеспечить кросс-платформенную совместимость.
- Сделать архитектуру прозрачной и расширяемой.
- Предоставить мощный CLI для управления задачами.

## ?? Установка и запуск

1.  **Сборка проекта:**
    ```bash
    cargo build
    ```

2.  **Запуск в стандартном режиме:**
    ```bash
    cargo run
    ```

3.  **Запуск в режиме отладки:**
    ```bash
    cargo run -- --debug
    ```

4.  **Просмотр справки по командам:**
    ```bash
    cargo run -- --help
    ```

## ?? Отладка и решение проблем
Если при сборке (`cargo build`) возникают ошибки, связанные с `Cargo.toml`, убедитесь, что:
- Все метаданные (description, authors) находятся в секции `[package]`.
- Зависимости перечислены только в одной секции `[dependencies]`.
